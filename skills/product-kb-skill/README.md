# product-kb-skill

Будущий skill базы знаний продукта.

## Ответственность

- поиск только по подтвержденным внутренним источникам;
- хранение документов, версий и source metadata в `kb-postgres`;
- индексация chunks в OpenSearch;
- возврат источников и событий через HTTP/SSE;
- отказ от финального ответа, если источники не найдены.

## Будущие Зависимости

- `kb-postgres`;
- `opensearch`.
