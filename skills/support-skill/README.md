# support-skill

Будущий skill для поиска по поддержке.

## Планируемые Источники

- support tickets;
- обращения пользователей;
- incidents;
- known problems;
- operational notes.

Skill должен помогать находить подтвержденные ответы по реальным обращениям, инцидентам и пользовательским проблемам.
